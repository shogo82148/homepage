svg2css

HOME>>ソフト>>svg2css

このソフトは何？

SVGで書かれた画像をHTMLとCSSに変換します。 このソフトを使えば、SVGに対応していないAndroid2.3以前のブラウザでも美しいベクター画像を楽しむことができます！

また、Pure CSS なスライドを作成することも可能です。

ダウンロード

Githubのsvg2cssからどうぞ。

サンプル

SVGからの変換サンプル。 Chrome, FireFox, Operaでの閲覧をお勧めします。 IE7以下では正常に表示できません。IE8以降は未確認。

ドロイド君

CSSなドロイド君 CSSドロイド君を別ウインドウで開く SVGドロイド君

QB

CSSなQB CSS QBを別ウインドウで開く SVG QB

スライド

発表用のスライドもPure CSSで実現できます。長岡技大LT部の発表で使いました。 スライドを別ウインドウで開く

アニメ

アニメーションだって再生できます。 アニメーションを別ウインドウで開く
